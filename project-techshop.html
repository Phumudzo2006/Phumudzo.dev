Phumudzo Tech Shop

A responsive e-commerce website showcasing technology products, built with modern HTML, CSS, and JavaScript.

Project Overview

Phumudzo Tech Shop is a responsive website designed to showcase tech products. It features a clean layout, smooth scrolling, interactive sections for services, projects, and contact information. Built to demonstrate web development skills and user-friendly design.

Key Features

  • Responsive layout for mobile and desktop
  • Interactive navigation with smooth scrolling
  • Services, projects, and contact sections
  • Downloadable resume integration
  • Portfolio ready for deployment on GitHub Pages
