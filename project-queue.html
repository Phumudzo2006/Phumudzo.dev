Public Transport Queue System (C++)

A console-based system managing passenger queues, priority handling, and reporting for transport systems.

Project Overview

This project simulates a public transport queue management system. It handles multiple vehicle types, passenger priorities, and provides detailed reports for boarding and departures.

Key Features

  • Queue management for different transport types
  • Priority boarding for passengers
  • Time-based departures and full vehicle handling
  • Console-based output for easy testing
  • Modular and reusable C++ code structure
