Transport Connectivity in Southern Africa (C++)

Analyzing regional transport networks using graph algorithms & connectivity metrics.

Project Overview

This project models the transport connectivity across Southern African cities using graphs. It uses BFS and Dijkstra to find paths, distances, and connectivity metrics.

Key Features

  • Adjacency matrix representation of city connections
  • BFS traversal for connectivity analysis
  • Dijkstra shortest path calculation
  • Distance, time, and cost metrics
  • Console-based reporting
