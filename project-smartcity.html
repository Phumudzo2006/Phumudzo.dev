Smart City Route Management System (C++)

A simulation system to manage city routes, find shortest paths, and make intelligent routing decisions.

Project Overview

This system models city networks and calculates optimal routes using algorithms like Dijkstra. It simulates route management for smart cities, providing distance, time, and cost metrics.

Key Features

  • Graph-based city network representation
  • Shortest path calculation using Dijkstra
  • Route simulation and logging
  • Console-based reporting
  • Flexible for adding more cities and connections
